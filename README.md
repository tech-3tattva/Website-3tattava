# Website-3tattava

This repository contains the full 3Tattva website codebase:

- `frontend/` - customer-facing Next.js application
- `backend/` - Express + MongoDB API
- `shared/` - shared TypeScript types

## Quick start

### Backend

```bash
cd backend
npm install
npm run seed
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
